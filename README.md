# redux-testing
Testing redux using thunks, promises, generators and async/await

Work in progress (i.e. not a thing yet)

To run the examples

### redux-thunk

```
npm run api
npm run basic-thunk
```


### redux-promise

```
npm run api
npm run basic-promise
```

Note: redux-promise-middleware example in the works



### redux-saga

```
npm run api
npm run basic-saga
```

## tests

```
npm run test
```
